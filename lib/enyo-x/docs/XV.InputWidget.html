Class: InputWidget

XV. InputWidget

An input control consisting of fittable columns: a styled label and an onyx.Input placed inside an onyx.InputDecorator, which provides styling.
Any controls in the InputDecorator appear to be inside an area styled as an input.
Use to implement a styled input field for strings.
Creates an HTML input element.
